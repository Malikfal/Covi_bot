## Название
Ковид-бот

## Описание
Бот-телеграм, помогающий узнать сколько человек сейчас болеет covid-19 в России и Китае

## Использованные технологии
Python3.9.13 :
    aiogram
    requests

## Запуск проекта
1. Скачайте проект
2. Откройте необходимую среду разработки
3. Запустите проект
